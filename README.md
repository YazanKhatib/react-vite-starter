# React Vite Starter

## Getting started

1. Gets the git project. (Auto run installation)

```
  npx @relateit/react-vite-starter my-app
```

2. Start the dev server. And open http://127.0.0.1:5173/ in your browser.

```
  yarn dev
```

3. Build your product.

```
  yarn build
```

## Dependencies

-   React
-   Vite
-   Typescript
-   eslint
-   Prettier
-   @apollo/client
-   Graphql
-   React router dom
-   i18next
